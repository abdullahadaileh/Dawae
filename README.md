# دوائي — Dawae 💊
## تطبيق إدارة الأدوية المنزلية الذكي

تطبيق ويب متجاوب لإدارة الأدوية المنزلية، مبني بـ HTML + CSS + JavaScript خالص مع Local Storage.

---

## ✨ المميزات

| الميزة | التفاصيل |
|--------|----------|
| 👤 تسجيل مستخدم متعدد | كل مستخدم له حساب وبيانات منفصلة |
| 💊 إضافة الأدوية | الاسم، الجرعة، الشكل، طريقة الاستخدام، الكمية، الصلاحية |
| ⏰ مواعيد التذكير | إضافة أوقات يومية متعددة لكل دواء |
| 📦 تتبع الكمية | شريط بصري لنسبة الكمية المتبقية |
| ⚠️ تنبيهات ذكية | تنبيه عند قرب النفاد أو انتهاء الصلاحية |
| 🔍 مقارنة الوصفة | مقارنة وصفة طبية جديدة مع مخزونك |
| 📊 لوحة التحكم | إحصائيات شاملة وتنبيهات في الصفحة الرئيسية |
| 📱 تصميم للهاتف | مُحسَّن بالكامل لشاشات الجوال |
| 💾 تخزين محلي | البيانات محفوظة في المتصفح (Local Storage) |

---

## 🚀 كيفية الاستخدام

### رفع على GitHub Pages

1. ارفع المجلد على GitHub
2. اذهب إلى **Settings → Pages**
3. اختر **Source: Deploy from a branch → main → / (root)**
4. انتظر دقيقة وسيظهر الرابط

### تشغيل محلي

ما تحتاج شيء — افتح `index.html` مباشرة في المتصفح.

---

## 📁 هيكل الملفات

```
dawae-app/
├── index.html    ← واجهة المستخدم الكاملة
├── app.js        ← المنطق والبيانات
└── README.md     ← هذا الملف
```

---

## 💾 آلية التخزين

كل مستخدم له مفتاح تخزين خاص:
```
dawae_users           → قاعدة بيانات المستخدمين
dawae_session         → جلسة المستخدم الحالي
dawae_meds_<email>    → أدوية كل مستخدم
```

---

## 🎨 التقنيات المستخدمة

- **HTML5** — هيكل التطبيق
- **CSS3** — تصميم متجاوب مع CSS Variables
- **Vanilla JavaScript** — منطق التطبيق بدون مكتبات
- **LocalStorage API** — التخزين المحلي
- **Google Fonts (Cairo)** — الخط العربي
